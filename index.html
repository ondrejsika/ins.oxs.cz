Install in one command, Debian only

Server Setup

cd && \
    apt update && \
    apt purge -y nano && \
    apt install -y vim mc htop git sudo curl wget && \
    mkdir -p .ssh && \
    curl -o .ssh/authorized_keys https://github.com/ondrejsika.keys
